Expressão regular não é um bicho de sete cabeças como as pessoas pensam. Na verdade é um dragão com oito. Hahahaha…​ Você 'programa' tudo em uma linha só.

+ Uma expressão regular é uma notação para representar padrões em textos. Serve para validar entradas de dados ou fazer busca e extração de informações em textos.

+ Um exemplo prático seria, por exemplo, localizar emails e telefones num texto muito grande usando um único comando na sua linguagem de programação.

Gerador de texto aleatório

Crie um mecanismo para obter texto ao mesmo tempo 1) aleatório e com 2) contexto definido.

Dessa forma será possível ter onde treinar o conhecimento de expressões regulares.

Use o console do Google Chrome como Playground.

  1. Acessar o google.com.br como site de busca.

  2. Faz uma pesquisa qualquer, por exemplo: quadrilatero.

  3. Copie e reserve o endereço da url carregada.

  4. Escreva a função abaixo. Use como base a url obtida:

    async function search(text) {
     const url = `https://www.google.com.br/search?q=${text}`;
     const response = await fetch(url);
     const content = await response.text();
     console.log(content);
     return content;
}

  5. Faça um teste pesquisando por um termo, por exemplo cabuloso

    1. Pesquise normalmente no google.com.br.

    2. Pesquise com a função search('cabuloso') e copie o conteúdo.

    3. Localize na internet um serviço tipo HTML Beauty, por exemplo o codebeautify.org.

    4. Formate o conteúdo e cole no editor.

Definição

Leia a definição.

Expressões regulares são padrões utilizados para selecionar combinações de caracteres em uma string. Em JavaScript, expressões regulares também são objetos. Elas podem ser utilizadas com os métodos exec e test do objeto RegExp, e com os métodos match, replace, search, e split do objeto String. Este capítulo descreve o uso de expressões regulares em JavaScript.

Criando uma Expressão Regular

Modo de construção Literal

Para expressões que não mudam

É compilada uma única vez quando o script é carregado.

Modo de construção String

Para expressões que podem mudar

É compilada em tempo de execução.

Exemplos
regex1 = /Brasil/;
regex2 = new RegExp('Brasil');
regex1b = /brasil/i;
regex2b = new RegExp('Brasil', 'i');

  • Explique as opções de flags (g, i, m) conforme a documentação de RegExp.

  • Demonstre erros com uso de flag errada.

  • Demonstre que cada instância é um objeto diferente, resultando em igualdade false. Parece String mas não é.

Expressão Regular no JavaScript

Analisando cada função que usa expressão regular. Por hora, focando mais no uso das funções do que na expressão regular. Usando expressões simples.

RegExp.prototype.exec vs. String.prototype.match

Tem resultado igual, mas a execução é feita a partir de referenciais opostos.

Tem como resultado um array com atributos a mais.

// Possuem o mesmo array como resultado.
match1 = /azul/.exec('o céu está azul');
match2 = 'o céu está azul'.match(/azul/);
RegExp.prototype.test

Verificação de true ou false para uma busca.

// Usado para verificação de Boolean.
if (/azul/.test('o céu está azul') === true) { console.log('ACHEI!'); }
String.prototype.search

Equivalente a String.prototype.indexOf, mas recebe um parâmetro uma que seja uma expressão regular. Caso não seja será convertida em uma.

Poupa múltiplas chamadas de String.prototype.indexOf.

text = 'o céu está azul';

// Três linhas com String.prototype.indexOf
positionCeu = text.indexOf('céu');
positionAzul = text.indexOf('azul');
firstPosition = positionCeu < positionAzul ? positionCeu : positionAzul;

// Uma linha com String.prototype.search
firstPosition = text.search(/(céu|azul)/);
String.prototype.replace

Hoje temos String.prototype.replaceAll, mas no passado JavaScript não tinha um método para substituir todas as ocorrências.

Confira no developer.mozilla.org os navegadores compatíveis e veja que nem todos aceitam ainda.

Mas usando String.prototype.replace com expressão regular alcançamos o resultado.

cpfCode = '123.456.789-00';

// Sem usar expressão regular
while (cpfCode.indexOf('.') >= 0) cpfCode = cpfCode.replace('.', '');
while (cpfCode.indexOf('-') >= 0) cpfCode = cpfCode.replace('-', '');

// Com expressão regular
cpfCode = cpfCode.replace(/[-.]/g, '');
String.prototype.split

Mais do mesmo. Reduz linhas de código.

datetime = new Date().toISOString();
dateTimeParts = datetime.split(/[^\d]/);
